```

│                          CortexPI System Architecture                            │
├───────────────────┬──────────────────────────────┬───────────────────────────────┤

│                   │                               │                               │
│  Apache Kafka     │  RAPIDS cuDF + Spark          │  FedAvg + Differential        │
│  Exactly-Once     │  Structured Streaming         │  Privacy across 5 Hospital    │
│  Semantics        │  Delta Lake (Lakehouse)        │  Nodes (torchrun dual-T4)     │
├───────────────────┴──────────────────────────────┴───────────────────────────────┤
│                            Deep Learning Core                                    │
│                                                                                  │
│   Mamba SSM ECG Encoder  ·  GRU-D Vitals Encoder  ·  Graph Attention Network    │
│   Custom Triton Pan-Tompkins GPU Kernel  ·  EWC Continual Learning              │
│   Conformal Prediction  ·  3 Clinical Prediction Heads                          │
├──────────────────────────────────────────────────────────────────────────────────┤
│   Datasets: PhysioNet 2012 · PTB-XL    Hardware: Apple M2 Pro · Kaggle dual T4  │
└──────────────────────────────────────────────────────────────────────────────────┘
```

- **Mamba SSM** encoder for long ECG sequence modeling — outperforms LSTM on irregular time series
- **GRU-D** vitals encoder with learned missingness decay for real ICU data sparsity

- **EWC continual learning** — prevents catastrophic forgetting as patient populations shift
- **Conformal prediction** — calibrated uncertainty bounds on all deterioration risk scores
- **2-GPU pipeline parallelism** via `torchrun` on Kaggle dual T4 instances
